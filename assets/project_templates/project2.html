Secure-Vault

🔍 Quick Summary

  • Type: Django Web Application for Password Security
  • Stack: Django, Python, HTML, CSS, JavaScript
  • Algorithms: Entropy calculation, Regex validation, AI-based strength logic
  • Duration: Feb 2025 – Mar 2025

📘 Description

Secure-Vault is a full-stack Django application that generates secure, high-entropy passwords and evaluates their strength in real-time using AI-based logic. It supports password criteria customization and educates users about safe password practices through intuitive UI feedback.

🚀 Key Features

  • AI-based Password Strength Checker with real-time feedback
  • Entropy scoring and regex-based validation
  • Clipboard-safe copy feature and secure generation
  • Interactive UI showing strength with animations

🧰 Tech Stack & Architecture

  • Frontend: HTML, CSS, JavaScript (Inline UI feedback)
  • Backend: Django, Python
  • Algorithms: Entropy Calculator, Regex Classifier
  • Deployment: PythonAnywhere / Render

Flowchart: User Input → Entropy Calculator → Strength Feedback → Suggestion Generator

🧠 Challenges & Resolutions

  • 💡 Ensuring accurate password strength metrics
  • ✅ Integrated entropy computation + AI-based dynamic feedback
  • 💡 Securing clipboard access
  • ✅ Implemented auto-clear logic and content encoding

💼 Job Roles Aligned

  • Django Developer
  • Security Software Developer
  • Application Security Engineer

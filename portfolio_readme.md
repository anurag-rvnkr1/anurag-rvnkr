## 🚀 **Deployment Guide**

### **Option 1: GitHub Pages (Recommended - Free)**
```bash
# 1. Create repository
git clone https://github.com/your-username/your-username.github.io.git

# 2. Add portfolio files
cp index.html your-username.github.io/
cd your-username.github.io

# 3. Deploy
git add .
git commit -m "Deploy hacker portfolio"
git push origin main

# 4. Access at: https://your-username.github.io
```

### **Option 2: Netlify (Free)**
1. Visit [netlify.com](https://netlify.com)
2. Drag and drop your HTML file
3. Get instant deployment with custom domain options
4. Automatic HTTPS and global CDN

### **Option 3: Vercel (Free)**
```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel --prod

# Custom domain available
```

### **Option 4: Traditional Web Hosting**
- Upload `index.html` to your hosting provider's public_html folder
- Compatible with any standard web hosting service
